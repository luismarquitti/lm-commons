# Integrations: MQT_Home Lab

## Core Services
- **Tailscale:** Used for remote access and inter-node communication without port forwarding.
- **GitHub:** Host for this repository and the Obsidian Vault.
- **Proxmox API:** Used by Ansible to provision and manage LXCs/VMs.

## External APIs
- **Google Gemini CLI:** Used by OpenClaw (requires OAuth).
- **Discord:** Used as a communication channel for OpenClaw agents.
- **Telegram:** Used for owner commands.
