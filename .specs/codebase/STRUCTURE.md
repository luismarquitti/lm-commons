# Structure: MQT_Home Lab

## Root Directory
- `ansible/`: Main IaC folder.
  - `inventory/`: Hosts and group variables.
  - `playbooks/`: Deployment scripts (numbered 00-09).
- `docs/`: Technical documentation and manual logs.
- `.specs/`: Spec-Driven Development (SDD) files.
  - `project/`: Vision, State, Roadmap.
  - `codebase/`: Brownfield mapping (this folder).
  - `features/`: Specific feature requirements and design.
  - `quick/`: Small/Ad-hoc task tracking.
- `files/`: Static files/templates for deployment.

## Key Files
- `GEMINI.md`: Project overview and context for AI agents.
- `CLAUDE.md`: Tool-specific guidance.
- `README.md`: High-level landing page.
